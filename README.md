# ZapretAutoTuner

Автоматический подбор рабочих стратегий и удобный графический интерфейс (GUI) для DPI-обходчика `zapret` под Windows.

Если вам надоело часами вручную ковыряться в консоли, подбирая рабочие комбинации параметров `desync`, `split` и `fooling` для обхода ограничений YouTube, Discord и сайтов, эта программа сделает всё за вас. Она сама протестирует пул комбинаций, измерит реальную скорость скачивания через жесткую SNI-блокировку и сохранит только те варианты, которые реально работают.



## Основные возможности

* **Два режима сканирования**: 
  * *Смарт-Подбор* - сверхбыстрый базовый перебор популярных рабочих параметров. Если они не сработают, скрипт автоматически перейдет в глубокий тест.
  * *Принудительный долгий подбор* - тотальный глубокий скан (Deep Scan) по расширенной матрице параметров для самых тяжелых случаев провайдерских блокировок.
* **Лог**: Вывод всех тестов в реальном времени с наглядной цветовой маркировкой (зеленый - успех, оранжевый - низкая скорость, красный - блок).
* **Встроенный менеджер конфигов**: Удобное окно со списком всех сохраненных батников. Позволяет в один клик перепроверить старую базу, замерить актуальную скорость и автоматически вычистить те конфиги, которые провайдер успел прикрыть.
* **Автоматический запрос прав**: Утилита сама запрашивает права администратора при запуске, так как без них работа низкоуровневого обходчика пакетов (`winws.exe`) невозможна.



## 📦 Как установить и запустить (Всё в одном)

> ⚠️ **Важно**: Данная сборка поставляется по принципу "всё включено". Главный скрипт `ZapPyEdition1.1.py` обязательно должен находиться в корневой папке `zapret` — строго рядом с каталогами `bin\` и `lists\`.
1. Откройте терминал в папке проекта и установите зависимости:
   pip install -r requirements.txt
